About the Exits column. The raw export reports the last item a child saw. Because a child who left after seeing item X actually exited the app when the next item appeared, every value has been shifted down by one position along the actual administration order (q01 → q02 → … → q10 → p01 → q11 → p02 → q12 → p03 → q13 → p04 → q14 → p05 → q15 → p06 → q16 → q17 → q18 → q19 → q20). We therefore cannot know how many children exited at q01 (the first scored item), and the 181 children whose last item was q20 are counted as having completed the assessment rather than as exits.
